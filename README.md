# Tab Groups Sidebar

A Chrome extension that provides a sidebar interface for managing Chrome tab groups.

## Features

- View all tabs organized by their tab groups
- Pinned tabs displayed separately at the top
- Color-coded groups matching Chrome's tab group colors
- Search tabs by title or URL
- Sort tabs by different criteria
- Keyboard shortcut: Alt+Shift+S (Windows/Linux) or Cmd+Shift+S (Mac)

## Permissions

This extension requires the following permissions:
- **tabs**: To read and display tab information
- **tabGroups**: To access tab group data
- **storage**: To save user preferences
- **sidePanel**: To display the sidebar interface
